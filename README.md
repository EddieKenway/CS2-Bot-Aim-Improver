# CS2-Bot-Aim-Improver
BotAimImprover is a plugin for CS2 (Counter Strike 2) built on [CounterStrikeSharp](https://docs.cssharp.dev/) that allows bots in practice mod aim for head instead of gut.
## Requirements
[Metamod:Source](https://www.sourcemm.net/downloads.php?branch=dev)  
[CounterStrikeSharp](https://docs.cssharp.dev/)
## Installation
1.Download the latest [releases](https://github.com/EddieKenway/CS2-Bot-Aim-Improver/releases)  
2.Extract the zip into **game/csgo/addons/counterstrikesharp/plugins** on your server  
3.Ensure the name of the folder is **BotAimImprover**  
4.Restart your server
## Commands
***mp_damage_headshot_only 1***  
Enable bots to aim for head.  
  
***mp_damage_headshot_only 0***  
Bots would aim for stomach.
